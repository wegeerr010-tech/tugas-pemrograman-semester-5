2. Alasan Pemilihan Widget

Column
Digunakan untuk menyusun komponen secara vertikal dari atas ke bawah sehingga tampilan lebih rapi.

Container
Digunakan untuk membuat kotak dengan warna, padding, border radius, dan shadow agar UI terlihat modern.

Text
Digunakan untuk menampilkan informasi workshop seperti judul, tanggal, lokasi, dan kuota.

ElevatedButton
Digunakan sebagai tombol daftar karena tampilannya jelas dan mudah digunakan pengguna.

Row
Digunakan untuk menyusun icon dan teks secara horizontal.

SingleChildScrollView
Digunakan agar halaman tetap bisa di-scroll jika isi workshop bertambah banyak.
